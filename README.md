# Akilli Ariza ve Saha Operasyon Platformu

Bu repo Clomosy TRObject ile gelistirilen yarisma projesi iskeletidir.

## Hedef
- Manuel takibi azaltmak
- Is emri kaybini onlemek
- Mudahale suresini kisaltmak
- Cozum suresinde %50 iyilesme hedeflemek
- Yoneticiye canli gorunurluk saglamak

## Teslim Sirasi
1. Mimari
2. Veri modeli
3. Ekranlar
4. MVP kapsami
5. Ilk modul secimi
6. Talep Acma Modulu TRObject iskeleti
7. Prosedurler giris/cikis mantigi
8. Hata kontrolleri
9. Test senaryolari
10. Sonraki modul onerisi

Detaylar docs klasorundadir.

## Kaynak Kod
- src/modules/talep-acma/talep_acma_module.tro

## Not
Bu surum MVP odakli Talep Acma modulunu calisir iskelet olarak sunar. Sonraki asamada Is Emri Havuzu ve Atama modulu eklenecektir.
