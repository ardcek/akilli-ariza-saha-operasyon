# 04 - Ilk MVP Kapsami

## MVP Dahil
1. Giris ve rol belirleme (basit)
2. Talep Acma Modulu
3. Is Emri Havuzu temel liste
4. Yonetici temel atama akisi
5. Saha personeli durum guncelleme (Atandi -> Yolda -> Islemde -> Tamamlandi)
6. Dashboard temel metrikleri

## MVP Disi (Sonraki Faz)
1. Gelismis rapor filtreleri
2. Coklu kurum/tenant destegi
3. Gelismis push bildirim senaryolari
4. Derin offline senkronizasyon motoru

## Basari Kriterleri
- Talep kaydi hatasiz acilabiliyor olmasi
- Atama ve durum gecisinin izlenebilir olmasi
- Dashboardda minimum 5 kritik metrik gosterimi

## Bu modulun eksikleri / sonraki adim
- MVP kapsamindaki servis sozlesmeleri (API contract) dokumani eklenecek
